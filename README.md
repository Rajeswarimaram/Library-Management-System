# Library Management System

## Project Overview

The Library Management System is developed using the ServiceNow platform. This application helps manage library books and student requests in a simple and organized way.

## Features

- Book Request
- Book Issue
- Book Return
- Student Record Management
- Book Availability Check
- Request Approval Workflow
- Notifications
- Reports

## Technologies Used

- ServiceNow
- Service Catalog
- Flow Designer
- Tables
- Forms
- Workflows
- Notifications

## How It Works

1. A student submits a book request.
2. The librarian reviews the request.
3. The request is approved or rejected.
4. If approved, the book is issued.
5. After use, the student returns the book.
6. The system updates the book status automatically.

## Benefits

- Easy to manage library records.
- Reduces manual work.
- Faster approval process.
- Keeps data organized.
- Improves library management.

## Conclusion

This Library Management System is built on the ServiceNow platform to simplify library operations. It provides an easy way to manage book requests, approvals, and returns while keeping all records organized.
